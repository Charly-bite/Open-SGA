Label Generation & Print Flow

A complete sequence of how a GHS label is generated — from user scanning a product code to the PDF appearing in the browser for printing.

sequenceDiagram actor User as 👤 Operator participant Browser as 🌐 Browser participant Flask as ⚡ Flask Route
(labels.py) participant Smart as 📦 SmartLabelManager participant Tara as ⚖️ TaraWeightManager participant GHS as 🏷️ GHSLabelGenerator participant RL as 📄 ReportLab participant P2I as 🖼️ pdf2image User->>Browser: Scan barcode / Enter product code Browser->>Flask: POST /labels/queue/add {code, qty} Flask->>Smart: get_product_data(code) Smart-->>Flask: {name, hazards, pictograms, H/P statements...} Flask->>Tara: resolve_tara(code, quantity) Tara-->>Flask: peso_tara (kg) Flask-->>Browser: ✅ Item added to session queue User->>Browser: Click "Print Labels" Browser->>Flask: POST /labels/generate_print {item_ids} Flask->>Smart: get_product_data(code) Smart-->>Flask: Full product data Flask->>GHS: generate_label(product_data, tmp.pdf) GHS->>RL: Build PDF (pictograms, H/P, batch, weight) RL-->>GHS: tmp.pdf saved GHS-->>Flask: PDF path Flask->>P2I: convert_from_path(tmp.pdf, 300 DPI) P2I-->>Flask: PNG image Flask-->>Browser: HTML print page with PNG images Browser->>User: 🖨️ Browser native print dialog
