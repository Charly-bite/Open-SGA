│ 💡 PROBLEMAS COMUNES Y SOLUCIONES                              │
├───────────────────────────────────────────────────────────────┤
│                                                                │
│  Problema: "Display not found" o "No module named _tkinter"  │
│  Solución: Tkinter no está instalado                         │
│    • Ubuntu/Debian: sudo apt install python3-tk              │
│    • Fedora/RHEL: sudo dnf install python3-tkinter           │
│    • Arch: sudo pacman -S tk                                 │
│                                                                │
│  Problema: Login aparece pero está en blanco                 │
│  Solución: Problema con Pillow                                │
│    pip install --upgrade Pillow                               │
│                                                                │
│  Problema: Console se queda en "Default admin account..."    │
│  Solución: SOLUCIONADO con la nueva versión                  │
│    Actualiza los archivos y reinicia                         │
│                                                                │
│  Problema: "Password incorrect" con admin/admin123           │
│  Solución: users.json puede estar corrupto                   │
│    rm users.json                                              │
│    python3 ghs_label_gui.py                                  │
│                                                                │
